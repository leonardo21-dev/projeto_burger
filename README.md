# 🍔 Burger - Landing Page de Hamburgueria

Projeto de uma página web responsiva para uma hamburgueria artesanal, desenvolvido com foco em layout moderno, organização de conteúdo e boas práticas de HTML e CSS.

## 📌 Sobre o projeto

A página apresenta:

- Seção inicial (hero) com destaque da marca
- Cardápio de hambúrgueres com descrição e preços
- Seção de bebidas
- Layout responsivo adaptado para diferentes telas
- Footer com informações de contato

O objetivo foi criar uma interface visual atrativa e organizada, simulando um cardápio digital.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Font Awesome

---

## 🎨 Layout e Estrutura

### 🔹 Header
Menu de navegação com links principais:
- Início
- Promoções
- Pedidos
- Conta

---

### 🔹 Seção Principal
- Título da marca
- Texto descritivo
- Imagem ilustrativa

---

### 🔹 Cards de Produtos (Cardápio)

Cada card contém:
- Imagem do produto
- Nome do hambúrguer
- Descrição
- Preço destacado

📌 Destaque técnico:
- Uso de **Flexbox** para manter o botão de preço sempre alinhado na parte inferior do card
- Estrutura responsiva com **CSS Grid**

---

### 🔹 Seção de Bebidas
Cards horizontais com:
- Imagem
- Nome da categoria de bebida

---

### 🔹 Footer
Informações de contato e localização

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem responsiva utilizando media queries:

- **Mobile First**
- Ajustes para tablets (`min-width: 37.5em`)
- Ajustes para desktops (`min-width: 48em` e `64em`)

---

## 🧠 Conceitos aplicados

- Organização semântica com HTML
- Reutilização de classes
- Layout com Grid (12 colunas)
- Alinhamento com Flexbox
- Uso de variáveis CSS (`:root`)
- Padronização visual (cores, espaçamento, tipografia)

## 🎯 Aprendizados

Durante o desenvolvimento foram trabalhados:

- Estruturação de layout responsivo
- Controle de alinhamento dinâmico (como manter elementos fixos dentro de cards)
- Separação de responsabilidades entre HTML e CSS
